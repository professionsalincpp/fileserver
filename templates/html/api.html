API Requests

This API supports the following requests:

Requests

  • Write
  • Read
  • Copy
  • Move
  • Create Directory
  • Create File
  • Simple Read File
  • Delete

Simple Read File

GET file HTTP/1.1

Read a file.

Parameter Type Description
path uri The path to the file to read from.

Write

PUT api/write HTTP/1.1

Write data to a file.

Parameter Type Description
path string The path to the file to write to.
data string The data to write to the file.
hash string The hash of the data to write.

Read

POST api/read HTTP/1.1

Read data from a file.

Parameter Type Description
path string The path to the file to read from.

Copy

POST api/copy HTTP/1.1

Copy a file.

Parameter Type Description
path.source string The path to the file to copy from.
path.destination string The path to the file to copy to.

Move

POST api/move HTTP/1.1

Move a file.

Parameter Type Description
path.source string The path to the file to move from.
path.destination string The path to the file to move to.

Create Directory

POST api/create/dir HTTP/1.1

Create a directory.

Parameter Type Description
path string The path to the directory to create.

Create File

POST api/create/file HTTP/1.1

Create a file.

Parameter Type Description
path string The path to the file to create.

Delete

DELETE api/delete HTTP/1.1

Deletes a file or directory.

Parameter Type Description
path string The path to the file or directory to delete.
